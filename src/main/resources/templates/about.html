Retrospective

This is a brief retrospective for this project.

My official start date for the project was 10/13/19 and I finished the project on 11/13/19. The project was broken into 3 phases. I was required to create a Trello board and complete this project using the Agile Methodology.

Phase 1 required me to create a Yahoo Finance account and save stocks to my portfolio. After this I was asked to create a console scraping application that would log into my finance account and scrape the data from the stock tables and write it to console.

Phase 2 required me to create a new database to work with. I was allowed to pick any database I wanted. After creating a database I would have to outline a schema to use for the stock data. I then had to integrate the newly created database into my stock scraper console app. The data needed to be stored in the database and persisted.

Phase 3 required me to implement a front end web application to showcase the scraper the data. I was required to implement user authentication, ensuring users could only access the data once they were registered in the database. I also needed to integrate my old console scraper into the web app and get it to work with the push of a button. Finally I was required to display the data from the current scrape, and also all previous scrape data.

For extra credit I decided to deploy my web scraper. I choose to use Heroku as the deployment platform. This required a great deal of troubleshooting and the web application you see here is the result.
